WAB showcase

A collection of Web applications created with Boundless Web App Builder (WAB) and WebSDK.

Original data, QGIS project and WAB appdef files are available in case you want to replicate this webapps yourself using WAB.

Symbology examples

Examples of advanced symbology, to show the large array of symbology elements that are supported by WAB

Arrows

Based on this blog post by Anita Graser, which describes how to create flow maps in QGIS. This symbology demonstrates WAB's support for expressions, data-defined parameters, geometry generators and arrow shapes.

Open Web App

Download QGIS project and appdef

OSM

An example that uses OSM data and a complex symbology to create proffesional-looking map. This example demonstrate line-following labels, support for symbol rendering levels (z-index), and rule-based rendering

Charts

An example that uses the charts widget to provide charting capabilities. The user can select features on the map and get a chart based on them. The example also includes other wigets such as geocoding, selection tools and attributes table.
